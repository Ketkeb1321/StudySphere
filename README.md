# 🌐 StudySphere

StudySphere is a smart school web platform that combines:

- 📚 Resource Hub (study materials)
- 💬 Anonymous Feedback System
- 🔍 Plagiarism Checker

Built using HTML, CSS, and JavaScript (no backend).

---

## 🚀 Features

### 📚 Resource Hub
- Subject-based materials (Math, Physics, Chemistry)
- Notes and video links

### 💬 Anonymous Feedback
- Submit feedback anonymously
- Stored using localStorage
- Dynamic display of messages

### 🔍 Plagiarism Checker
- Compares text with sample content
- Shows similarity percentage
- Highlights matching words

---

## 🧱 Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage
